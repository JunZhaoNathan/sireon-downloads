# Sireon Downloads

Sireon 是 Vellumloop 旗下的 macOS 本地优先 AI 个人规划工作台。它把自然对话、计划候选、计划块、连续计划、提醒、笔记、导入证据、个人节律和执行历史放进同一个实用循环。

This public repository only hosts Sireon macOS downloads. The application source code is private.

## 安装

1. 下载最新版 `.dmg` 文件。
2. 双击打开 DMG。
3. 把 `Sireon.app` 拖到 `Applications / 应用程序` 文件夹。
4. 从 `Applications / 应用程序` 打开 Sireon。

如果 macOS 提示无法打开：

1. 打开 `系统设置 -> 隐私与安全性`。
2. 在安全提示处选择“仍要打开”。
3. 或右键点击 `Sireon.app`，选择“打开”。

## 试用与激活

Sireon 的本地规划、笔记、提醒和执行记录能力可以直接使用。AI 功能需要试用激活码或正式激活码。

申请试用：

1. 打开 Sireon。
2. 进入 `设置 -> AI 服务 -> Sireon AI 授权`。
3. 复制页面里的“本机机器码”。
4. 到 Sireon 官网提交试用申请：

```text
https://sireon.vellumloop.com
```

5. 收到邮件中的激活码后，回到授权页面粘贴激活码并点击激活。

激活码会绑定本机机器码。换电脑后机器码会改变，需要重新申请或联系 Vellumloop 重新发码。

## 更新

发布新版本时，本仓库的 Releases 会更新。下载新版 DMG 后，重新把新的 `Sireon.app` 拖入 `Applications / 应用程序` 替换旧版本即可。

原来的激活码通常可以继续使用，只要它没有过期，并且仍在同一台机器上使用。

## English

Sireon is a local-first AI personal planning workbench for macOS. It connects natural chat, planning candidates, plan blocks, continuous plans, reminders, notes, imported evidence, personal rhythm, and execution history in one practical loop.

Download the latest macOS DMG from:

https://github.com/JunZhaoNathan/sireon-downloads/releases/latest

To install, open the DMG, drag `Sireon.app` into `Applications`, and launch it from there. If macOS blocks the first launch, open `System Settings -> Privacy & Security` and allow the app, or right-click the app and choose `Open`.

AI features require a trial or activation code. Copy the machine code from `Settings -> AI Service -> Sireon AI License`, submit a trial request at `https://sireon.vellumloop.com`, then paste the activation code you receive by email into the license page.
